I. Алгоритм работы квантования

1) Поточний час встановлюється в початковий час підйому.
2) Створюється масив для запису поточних індексів кожного з сигналів(не пам'ятаю навіщо потрібно це робити взагалі).
3) В масиві масивів квантованих значень часу та датчиків сторюються пусті масиви у кількості яка відповідає кількості сигналів.
4) Використовується цикл for для перебору масивів сигналів та часу
4.1) Використовується while для проходження по елементам масиву до stoptime
4.1.1) For використовується для підгонки currentTime та проходження по масиву значень та запису квантованих даних
4.2) По досягненню stoptime, currentTime встановлюється в startTime і починається проходження по наступному масиву сигналів
